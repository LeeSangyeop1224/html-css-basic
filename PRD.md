## 페이지 구성

---

### 로그인 페이지

- 로그인 화면 표시 로그인 인증이 완료되면 매인 페이지로 이동

---

### 메인 페이지

- 화면에 비행드론 기록, 기록된 정보 열람하기 선택지 생성
- 선택에 따라 현재상황 페이지나, 기록 목록 페이지로 이동

---

### 현제 페이지

- 현재 비행중인 드론과 연결하여 고도, 방위, GPS등의 정보를 화면에 표출

---

### 기록 페이지

- 시간, 날자, 드론기종이 적힌 목록표를 생성 원하는 기록의 제목을 선택하면 해당정보를 재생
